## Touch Interaction Specs

### Tap Targets
- Minimum size: 44x44px
- Spacing: 8px minimum
- Visual feedback: 150ms highlight

### Gestures
- Swipe right: Show navigation
- Swipe left: Hide navigation
- Pull to refresh: Update chat
- Long press: Context menu
